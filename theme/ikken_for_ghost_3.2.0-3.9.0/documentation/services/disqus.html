Disqus comments

After you’ve added your site to Disqus, you will be asked to choose a platform.
Select Universal Code from the list of options.

Now that you have your Universal Code copy only your Shortname (example: your-site.disqus.com - copy only your-site). You’ll need to locate and open the theme’s ../partials/services/disqus.hbs file in your HTML editor.

Paste your Shortname - the only thing you need to change is XXXXX - which should be replaced with your own Disqus site-id) and remove brackets {{!-- --}} to use Disqus comments.

Save your disqus.hbs file in ../partials/services/ folder.

Congratulations! You should now have a Disqus comment box.
