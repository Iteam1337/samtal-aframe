Featured label

The label can be customized using adequate tags. By default, the label displays all information.

Global options

Hiding the publish date

To hide publish date just add an adequate tag #label-date-hide in Post Settings menu to one of the posts.

Hiding the icon

To hide icon just add an adequate tag #label-icon-hide in Post Settings menu to one of the posts.

Hiding the “small” text (by default “of the month”)

To hide small text just add an adequate tag #label-small-hide in Post Settings menu to one of the posts.

Small icon variant

To activate the smaller icon variant just add an adequate tag #label-icon-small in Post Settings menu to one of the posts.

Disable all featured labels

To disable all Featured labels just add an adequate tag #label-disable in Post Settings menu to one of the posts.

Hiding the label for a specific post

To hide the label for a specific post just add an adequate tag #label-post-hide in Post Settings menu to the post.

How to add translate?

All texts can be found in the .json files in ../locales folder.
Please read chapter Theme Translation.
