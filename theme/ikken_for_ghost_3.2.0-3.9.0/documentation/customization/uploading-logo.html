Logo

Uploading

Select General from the navigation menu on the left of your admin area.

Then scroll down and at the bottom you will see an option which says Publication logo.
Click Upload Image to upload your logo.

Click Save Settings to activate.

Congratulations! Your logo is active on the blog.

Changing the size

By default logo height is 45px.

To change to another value select Code injection from the navigation menu on the left of your admin area.

And paste this code to Site Header:

<style>
.header-logo img {
   max-height: 45px;
}
</style>

Change the value of the max-height: 45px parameter to the expected value.

Click Save to activate.
