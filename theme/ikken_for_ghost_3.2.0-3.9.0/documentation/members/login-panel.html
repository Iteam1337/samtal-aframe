Login panel in navigation

By default - when Members function is active, login panel it’s enabled. To disable just add an adequate tag #login-panel-disable in Post Settings menu to one of the posts.

Slug URL must be hash-login-panel-disable

Replacing “Sign up” with “Membership”

You can easily replace the Sign up field with Membership.

  1. For this purpose open ../partials/members_login_panel.hbs file in your HTML editor and find:
<a class="members-signup" href="{{@site.url}}/signup/">{{t "Sign up"}}</a>
  1. Then replace with:
<a class="members-signup" href="{{@site.url}}/membership/">Membership</a>
  1. Add membership page with /membership/ URL slug.
